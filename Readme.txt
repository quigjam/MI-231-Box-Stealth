MI 231 - Custom Game Project
Box Stealth - James Quigley
""""""
3d stealth game where the player tries to get to the other side of the level with out being detected
""""""
Scenes
""""""
Scene_0 - Main Menu
Scene_1 - Level Selector
Scene_2 - Level 1
Scene_3 - Level 2
""""""
Controls
""""""
WASD/Arrows - move player
Mouse cursor - direct palyer line of sight (dosen't have any really effect on gameplay)
C - crouch (make yourself smaller, move quieter)
P - pause (options menu and freeze game, can restart or go to main menu from here)
""""""
Features
""""""
PlayerController with full range of movement
Player is able to crouch
Enemy script is the main focus of alot of features
robust detect system for "seeing" and "hearing" the player
variance in level of detect based on player proximity/exposure
Enemy will look around for player if they are detected enough
Light cones to give better visualization of enemy field of view
Camera enemy which stays stationary but pivots to patrol an area
Waypoint movement for Enemies used NavMesh
Options menu and volume control which saves between scenes
